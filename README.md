Complaint Management System

A simple Java Swing based Complaint Management System with a modern UI design.
This project allows users to:

- Login to the system
- Register complaints
- Generate complaint IDs
- Check complaint status
- Use Forgot Password functionality

The application is built using Java Swing and is designed as a beginner-friendly desktop project.

---

Features

User Login

- Username and password login interface
- Modern colorful UI inspired by modern apps

Complaint Registration

Users can:

- Select complaint type
- Enter complaint description
- Submit complaints

Complaint ID Generator

- Automatically generates a unique complaint ID
- Example:

CMP4821

Complaint Status Checker

Users can:

- Enter Complaint ID
- View complaint status

Forgot Password

- Recovery popup for password reset simulation

Modern UI

- Soft background colors
- Styled buttons
- Better fonts
- Centered windows
- Scrollable text area

Step 3

Run the program
